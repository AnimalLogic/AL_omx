OMX
=======

Friendly and Fast Maya Python API.

OMX tries to make OM2 easier to use but still fast.

This presentation highlights some of the reasoning and how OMX works:
`OMX Presentation <https://github.al.com.au/pages/aloysb/presentations/omx>`_

See :py:mod:`AL.maya2.omx` for reference docs.
